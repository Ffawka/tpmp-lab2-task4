# TPMP Lab2 Task4 - File Processing

## Overview
Программа для обработки текстовых файлов на языке C.
Выполняет:
- Создание текстового файла с информацией
- Просмотр содержимого файла
- Чтение и обработку данных из файла
- Сохранение результатов в новый файл

## Usage
1. Сборка проекта: `make`
2. Запуск программы: `./program`
3. Очистка: `make clean`

Программа создает входной файл `data/input.txt`, обрабатывает его и сохраняет результат в `data/output.txt`.

## Additional Notes
Проект демонстрирует работу с файлами в C и следует модели КИС.
